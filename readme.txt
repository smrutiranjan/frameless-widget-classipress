== Description ==

Adds a text-like widget that allows you to write Frameless in it. (Just whats missing in the default text widget)

== Installation ==

1. Add the plugin's folder in the WordPress' plugin directory.
1. Activate the plugin.
1. You are now ready to use the Frameless Widget from the Widgets section.
